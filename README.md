# SignalHub Enterprise Portal

Enterprise-style partner operations and revenue intelligence portal built as a portfolio project for cloud, applications, integration, and architecture-oriented roles.

## Initial Structure
- `src/` application projects
- `tests/` automated tests
- `infra/terraform/` infrastructure as code
- `docs/architecture/` system and technical design docs
- `docs/runbooks/` support and operational runbooks
- `docs/adr/` architecture decision records
- `scripts/` helper scripts
